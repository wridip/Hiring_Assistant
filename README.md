# TalentScout Hiring Assistant - Run Guide

This README provides instructions on how to set up and run the TalentScout Hiring Assistant locally on your machine.

## Prerequisites

1.  **Python 3.10 or higher**: Ensure Python is installed. Check using `python --version`.
2.  **Ollama**: This project uses **Llama 3** running locally. 
    - Download and install Ollama from [ollama.com](https://ollama.com/).
    - After installation, open your terminal and run:
      ```bash
      ollama pull llama3
      ```

## Installation

1.  **Clone the Repository**:
    ```bash
    git clone <repository_url>
    cd Hiring_Assistant
    ```

2.  **Install Dependencies**:
    It is recommended to use a virtual environment.
    ```bash
    # Create virtual environment
    python -m venv venv
    
    # Activate virtual environment (Windows)
    .\venv\Scripts\activate
    
    # Activate virtual environment (Mac/Linux)
    source venv/bin/activate

    # Install the required packages
    pip install -r requirements.txt
    ```

## Running the Application

1.  **Start Ollama**: Ensure the Ollama application is running in the background.
2.  **Launch Streamlit**:
    ```bash
    streamlit run app.py
    ```
3.  **Access the Chatbot**: Once the command is executed, your default browser will open to `http://localhost:8501`. If it doesn't open automatically, you can manually enter the URL.

## Using the Chatbot

- The chatbot will prompt you for your details sequentially.
- Follow the instructions and provide your tech stack when asked.
- Answer the technical questions generated by the AI.
- To end the chat, type `bye`, `exit`, `quit`, or `thanks`.

## Project Structure
- `app.py`: Main application code (Streamlit UI).
- `requirements.txt`: Python dependencies.
- `utils/`: Logic for validation and LLM communication.
- `prompts/`: Text templates for the AI model.
- `data/`: Location where candidate profiles are saved.
