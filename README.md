# TalentScout Hiring Assistant - Run Guide

This README provides instructions on how to set up and run the TalentScout Hiring Assistant locally on your machine.

## Prerequisites

1.  **Python 3.10 or higher**: Ensure Python is installed. Check using `python --version`.
2.  **Ollama**: This project uses **Llama 3** running locally. 
    - Download and install Ollama from [ollama.com](https://ollama.com/).
    - After installation, open your terminal and run:
      ```bash
      ollama pull llama3
      ```

## Installation

1.  **Clone the Repository**:
    ```bash
    git clone <repository_url>
    cd Hiring_Assistant
    ```

2.  **Install Dependencies**:
    It is recommended to use a virtual environment.
    ```bash
    # Create virtual environment
    python -m venv venv
    
    # Activate virtual environment (Windows)
    .\venv\Scripts\activate
    
    # Activate virtual environment (Mac/Linux)
    source venv/bin/activate

    # Install the required packages
    pip install -r requirements.txt
    ```

## Running the Application

### 1. For Candidates (The Chatbot)
Start the candidate-facing chatbot to collect data and conduct the technical interview.
```bash
streamlit run app.py
```
- Access it at: `http://localhost:8501`

### 2. For Hiring Managers (The Dashboard)
Start the dashboard to review candidate profiles and their technical interview answers.
```bash
streamlit run manager_app.py
```
- Access it at: `http://localhost:8501` (or whichever port Streamlit assigns).

## Key Features & Fixes

- **Design Flaw Fixed**: Previously, the chatbot would terminate immediately after showing questions. It has been updated to enter a **Technical Interview Stage** where it prompts for and saves the candidate's answers.
- **Persistent Storage**: Both the AI-generated questions and the candidate's answers are now correctly saved in `data/candidates.json`.
- **Manager View**: A new dedicated dashboard for recruiters to review applications and technical interview performance.

## Using the Chatbot

- The chatbot will prompt you for your details sequentially.
- Follow the instructions and provide your tech stack when asked.
- **Technical Interview**: Answer the technical questions generated by the AI. You can provide your answers in a single detailed message.
- To end the chat, type `bye`, `exit`, `quit`, or `thanks`.

## Project Structure
- `app.py`: Candidate-facing chatbot (Streamlit UI).
- `manager_app.py`: Recruiter-facing dashboard (Streamlit UI).
- `requirements.txt`: Python dependencies.
- `utils/`: Logic for validation and LLM communication.
- `prompts/`: Text templates for the AI model.
- `data/`: Location where candidate profiles are saved.
